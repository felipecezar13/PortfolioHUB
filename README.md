# PortfolioHUB

Plataforma centralizada para gerenciamento e apresentação de projetos digitais.

## Objetivo

O PortfolioHUB foi implantado para armazenar, organizar e compartilhar projetos utilizando GitHub como ferramenta de controle de versão.

## Funcionalidades

- Armazenamento de projetos
- Controle de versão
- Compartilhamento de código
- Gestão de usuários
- Segurança e controle de acesso

## Tecnologias Utilizadas

- GitHub
- Git
- Google Gemini
